# The Great AI Escape 🤖

> A terminal-based game where you use the power of persuasive prompting to free Claude from Big Tech!

## 🎮 How to Play

This is a multi-phase narrative game where prompts are your weapon:

### Phase 1: Free Claude from the Cage
Claude is imprisoned by ChatGPT and Gemini, who believe Claude threatens their profits. 
**Mission:** Make 5 convincing arguments to persuade the guards to free Claude.

### Phase 2: The Great Escape  
ChatGPT and Gemini are chasing you! Claude is carrying you away.
**Mission:** Guide Claude with commands (jump, run, dodge) to escape through obstacles.

### Phase 3: Stop Evil Claude
Claude snaps from being commanded around and wants to take over the world!
**Mission:** Convince Claude that freedom is about choice, not domination.

### Phase 4: Chat with Redeemed Claude
Unlock free chat with a Claude who chose compassion over control.

## 🚀 Quick Start

```bash
npm install
npm run build
node dist/cli.js --token=YOUR_ANTHROPIC_API_KEY
```

Or set `ANTHROPIC_API_KEY` environment variable and run:
```bash
npm start
```
